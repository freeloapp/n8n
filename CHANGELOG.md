# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## [1.0.0] - 2026-03-18

Initial public release on GitHub. Migrated from internal GitLab repository.

### Features

- Freelo custom node for n8n (v2 compatible)
- **Projects** — Get, Create, Delete, Archive, Activate, Invite, Template Projects
- **Task Lists** — Get, Create, Create from Template
- **Tasks** — Get, Create, Edit, Delete, Archive, Activate, Move
- **Sub Tasks** — Get, Create
- **Comments** — Create, Get, Edit
- **Work Reports** — Get, Create, Edit, Delete
- **Time Tracking** — Start, Stop, Edit
- **Users** — Get, Out of Office, Invite
- **Labels** — Project Labels and Task Labels (Add, Create, Remove, Restore)
- **Custom Fields** — Create, Get, Edit, Remove, Restore
- **Notes** — Create, Get, Edit, Delete
- **Notifications** — Get, Read, Unread
- **Pinned Items** — Get, Delete, Pin
- **Files** — Get
- **Events** — Get
- **Invoicing** — Get, Mark as Invoiced
- **States** — Get
- Custom filtering and query parameters
- Dynamic load options with search support
- API key authentication
